Facility Cards

Facility cards can be created via two different markups. Standard markup is supported via theme-mylt/_card.scss, and represents the way these cards function within the facilities schedules. Notecard markup is intended to demonstrate the framework tools built into theme-base/_card-notecard.scss and the addition of color classes via theme-base/_card-suits.scss if you wish to recreate the look of these cards for other use cases.

Please note that margins have switched to be entirely margin-bottom based for consistency in the Notecard version. I've used margin-bottom on the headings to equalize this difference in these examples.
