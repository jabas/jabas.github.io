Headings

All HTML headings, <h1> through <h6>, are available. .h1 through .h6 classes are also available, for when you want to match the font styling of a heading but still want your text to be displayed inline. A heading with .h-th is a special case that styles like a table header cell.

Headings utilize sass maps to make property changes per breakpoint to keep sync with the vertical rhythm and maintain comfortable reading.
