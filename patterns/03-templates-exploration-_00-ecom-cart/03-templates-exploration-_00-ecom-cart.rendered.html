CSS

Important to note: this SCSS is written as an override to what exists currently in the styles. When adapting this, some of this will mean you need to delete existing LESS code instead of adding to it. The finished output should match what appears here, however. This now uses flexbox css for layout.

Also, not being familar with their codebase, I opted to keep the level of selector depth present in the existing LESS, which has longer selectors than I'd prefer.
