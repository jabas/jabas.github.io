Carousels

Default Carousel

This is the basic Bootstrap 4 carousel, using the .carousel class. Carousel indicators space themselves out more in XS and SM breakpoints, so long as there are 5 or less items in the carousel.

Please note: the version of Bootstrap we are using has a broken keyboard navigation feature enabled. Any instance of the carousel must include the data-keyboard="false" attribute.
