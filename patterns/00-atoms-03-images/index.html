SVG Embedded Artwork

SVG art can be embedded via an external SVG file. Because these use the same <use> method as our icon library, they can similarly respond to color changes via currentColor. Additionally, resizing this artwork will not change the stroke width; it will remain 1px regardless. As these images are 72x72, you may see anti-aliasing effects if you don't use increments of 72 (or 36 should work decently enough, especially on retina screens).
