Brightcove Video

Brightcove Inline Video Player

Life Time uses Brightcove to host and deliver the majority of our videos. To use the inline player, you will need to ensure the Brightcove javascript at //players.brightcove.net/5121028912001/ryt7Y2djx_default/index.min.js is included. An empty video element with data attributes can then be read by their script to import the video file and player. Pair this with the .embed-responsive classes to make the video responsive.
