Card Carousel

This carousel will display a different number of cards at each breakpoint, and change the UI accordingly for the number of slides required. To use, have each individual .card or .card-feature in its own .carousel-item to match the XS breakpoint, and the accompanying JavaScript will alter the layout for higher breakpoints at document ready, or when dropped on the page after ready. There is also a debounced resize watch to change the layout as needed.

Currently, the carousel will reset itself to the first slide as active on any layout change, including that when loaded.

Please note: the current version will keep cards the same height per slide, but not across multiple slides. If all cards are not of uniform height, you will see the carousel adjust in height as you advance through it.
